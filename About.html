About OpsTrack

OpsTrack is a front-end web project designed to support small business decision-making through organized operational data.

Project Purpose

Many small businesses need quick access to important information like revenue, labor, customer trends, and booking activity. OpsTrack provides a dashboard-style layout that makes those metrics easier to understand. This project was created as a CIS course project using HTML, CSS, and JavaScript, with the potential for future back-end integration.

Built With

HTML for structure, CSS for design, and JavaScript for interactivity.

Target Users

Small business owners and managers who need performance visibility in one place.

Future Goals

Add user accounts, data storage, reports, and live operational tracking.
